## Summary

The tweet argues that treating memory as an action in React-like frameworks, allowing LLMs to autonomously decide what to store/retrieve during reasoning, could render current memory layers for agents obsolete if big model companies adopt this approach. It critiques existing memory systems as outdated without proper training strategies.
