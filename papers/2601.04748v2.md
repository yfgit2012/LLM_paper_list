## Summary
- **Objective**: To address challenges in multi-agent systems (MAS) by compiling them into single-agent systems (SAS) using skill libraries, improving performance and reducing cognitive load while enabling scalable, cognitively efficient systems.  
- **Methodologies**:  
  - **MAS-to-SAS Compilation Algorithm**: Extracts and classifies agent capabilities into skills, internalizes interaction constraints, and transforms MAS into SAS.  
  - **Skill Library Design**: Structured skill libraries with descriptors, execution policies, and constraints for skill selection.  
  - **Empirical Evaluation**: Tests performance improvements, cognitive load reduction, and scalability via experiments with skill libraries, tasks, and prompt templates.  
- **Results**:  
  - Compilation improves performance and reduces cognitive load, particularly in complex environments.  
  - Skill libraries exceeding a capacity threshold (κ) necessitate hierarchical multi-agent organization to mitigate cognitive overload.  
  - Semantic confusability (similar skill phrasing) impacts performance, and policy complexity (flat vs. hierarchical) affects task execution.  
- **Key Contributions**:  
  1. A systematic MAS-to-SAS compilation algorithm.  
  2. A structured skill library design integrating descriptors, policies, and constraints.  
  3. Empirical validation of performance gains and cognitive load reduction.  
  4. Theoretical connections to cognitive psychology and scaling laws, highlighting hierarchical organization as a necessity for large skill libraries.  
- **Conclusions**: Hierarchical multi-agent organization is not merely an architectural choice but a required mitigation when skill libraries surpass a critical capacity threshold. The work bridges MAS and SAS, offering insights into cognitive limits and scaling behavior for scalable systems.

## 摘要
- **目标**：通过使用技能库将多智能体系统（MAS）编译为单智能体系统（SAS），解决MAS中的挑战，提升性能、降低认知负荷，并实现可扩展且认知高效的系统。  
- **方法**：  
  - **MAS到SAS编译算法**：提取并分类智能体能力为技能，内化交互约束，并将MAS转换为SAS。  
  - **技能库设计**：结构化的技能库，包含描述符、执行策略及技能选择约束。  
  - **实证评估**：通过技能库、任务和提示模板实验，测试性能提升、认知负荷降低和可扩展性。  
- **结果**：  
  - 编译显著提升性能并降低认知负荷，尤其在复杂环境中。  
  - 超过容量阈值（κ）的技能库需采用分层多智能体组织以缓解认知过载。  
  - 语义混淆度（相似技能表述）影响性能，策略复杂度（扁平 vs. 分层）影响任务执行。  
- **关键贡献**：  
  1. 系统化的MAS到SAS编译算法。  
  2. 集成描述符、策略和约束的结构化技能库设计。  
  3. 实证验证性能提升和认知负荷降低。  
  4. 与认知心理学和扩展定律的理论联系，强调大规模技能库需分层组织。  
- **结论**：分层多智能体组织不仅是架构选择，更是当技能库容量超过临界阈值时的必要缓解措施。该工作连接MAS与SAS，为可扩展系统的认知限制和扩展行为提供洞见。
