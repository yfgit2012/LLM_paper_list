## Summary  
- **Objective**: To infer the original instruction from a response by analyzing differences between a "chosen" (correct) and "rejected" (incorrect) response, enabling accurate reconstruction of the original task specification.  
- **Methodologies**: FLIP employs contrastive analysis by comparing chosen and rejected responses to identify critical elements (e.g., formatting, calculation steps, content requirements). It reconstructs the original instruction by incorporating these elements and validates accuracy using F1 scores.  
- **Results**: FLIP achieves higher F1 scores for inferred instructions derived from chosen responses compared to rejected ones. For example, in math tasks, converting answers to 10⁹ multiples improved F1 scores (0.419 vs. 0.375), and adding emojis in precise IF tasks increased scores (0.370 vs. 0.174).  
- **Key Contributions**:  
  - Introduces FLIP as a novel method for instruction inference via contrastive analysis.  
  - Highlights precision in identifying task-specific requirements (e.g., formatting rules, calculation steps).  
  - Demonstrates adaptability across diverse tasks (math, definition, explanation) and efficiency by avoiding model retraining.  
- **Conclusions**: FLIP effectively reconstructs original instructions by focusing on explicit differences between correct and incorrect responses, improving inference accuracy and offering a scalable approach for refining task specifications.
